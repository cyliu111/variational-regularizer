Pytorch implementation for std regularization.
Testing on Pytorch1.1.0, python3.6

Reference:
Deep Convolutional Neural Networks with Spatial Regularization, Volume and Star-shape Prior for Image Segmentation.
https://arxiv.org/abs/2002.03989


This demo is only a forward implementation for std.


1. How to run the demo?
python demo_std.py

2. One can replace it to the softmax activatiation in any segmentation CNNs to guarantee smooth boundaries.
